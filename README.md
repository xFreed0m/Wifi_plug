# Wifi plug

Simple python2 script to automate installation of wifi pentesting tools.

## How to use it
First, install the needed dependencies:
```
pip install -r requirments.txt
```
Second, run the tool:
```
python wifi_plug.py
```
 
### Compatibility
This was tested with python 2.7 and Kali 2019.2
